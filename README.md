## Отчет по практической работе №6 (Часть 3)
Тема: Создание автоматизированных Unit-тестов модулей регистрации и авторизации.
Выполнили: Студенты группы 3ИСИП-423 Соболенко А.Р., Новиков Д.Е.

В ходе данной практической работы был проведен рефакторинг кода и созданы методы для тестирования авторизации и регистрации пользователей в нформационной системе кинотеатра. Были рассмотрены как позитивные, так и негативные тесты.

### Вывод о проведенном тестировании
В ходе выполнения практической работы были разработаны автоматизированные тесты для методов Auth и Registration. По результатам запуска тестов было выявлено, что успешно прошли негативные тесты авторизации и тесты капчи. Все остальные тесты завершились негативно.
Тесты могли пройти негативно из-за отсутствия соответствующих проверок непосредстенно в программе, в то время как позиитивные тесты наоборот показывают, что эти уязвимости были учтены и проработаны при разработке.
